ImageWriter#

saveWriterCreateEmpty#

int saveWriterCreateEmpty(saveWriter *phWriter)#

saveWriterCreateEmpty creates an empty/default image writer. The parameters of this image writer are not set (height = width = bits per pixel = 0). The default file name pattern is ‘savedimages/image<count>.jpg’ and the default file format is jpeg.

Warning

  • unable to save before setting parameters

Parameters:

phWriter –

  • Type: saveWriter*

  • [Out] parameter

  • Created image writer

Returns:

  • Type: SC_ERROR

  • Error code for the function

  • Returns SC_ERR_SUCCESS (0) on success

saveWriterSetPlyAndConfigExtended#

int saveWriterSetPlyAndConfigExtended(saveWriter hWriter, savePlyParams params)#

saveWriterSetPlyAndConfigExtended changes the image writer to save images as ply data. It switches the extension to ‘.ply’ and uses savePlyParams information to configre the save data. The param.filterPoints option will filter NaN points in the output data. The params.scale, and params.offset(A/B/C) will be applied to the output PLY file accordingly also. Finally the params.isSigned value lets the library know if data passed in is signed or unsigned.

Parameters:

  • hWriter –

    • Type: saveWriter

    • [In] parameter

    • An image writer

  • params –

    • Type: savePlyParams,

    • [In] parameter

    • Passes savePlyParams to saveWriterSetPlyAndConfigExtended

Returns:

  • Type: SC_ERROR

  • Error code for the function

  • Returns SC_ERR_SUCCESS (0) on success
