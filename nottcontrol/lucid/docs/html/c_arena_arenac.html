acOpenSystem#

AC_API acOpenSystem(acSystem *phSystem)#

acOpenSystem initializes the Arena SDK and retrieves the system object (acSystem). The system must be closed, or memory will leak.

See also

  • acSystem

Parameters:

phSystem –

  • Type: acSystem*

  • [Out] parameter

  • The system object

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success

acCloseSystem#

AC_API acCloseSystem(acSystem hSystem)#

acCloseSystem cleans up the system (acSystem) and deinitializes the Arena SDK, deallocating all memory.

See also

  • acSystem

Parameters:

hSystem –

  • Type: acSystem

  • [In] parameter

  • The system object

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success

acGetBitsPerPixel#

AC_API acGetBitsPerPixel(uint64_t pixelFormat, size_t *pBitsPerPixel)#

acGetBitsPerPixel gets the number of bits per pixel of the image from the integer value of the pixel format (PfncFormat).

Parameters:

  • pixelFormat –

    • Type: uint64_t

    • [In] parameter

    • Pixel format to get bits per pixel

  • pBitsPerPixel –

    • Type: size_t*

    • [Out] parameter

    • Number of bits per pixel for given pixel format

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success

acCalculateCRC32#

AC_API acCalculateCRC32(const uint8_t *pData, size_t pDataLen, size_t *pCRCValue)#

acCalculatedCRC32 calculates a CRC value (cyclical redundancy check) on a dataset. This is used to check whether the dataset has been sent in its entirety.

Parameters:

  • pData –

    • Type: const uint8_t*

    • [In] parameter

    • A pointer to the data to use to calculate the CRC

  • pDataLen –

    • Type: size_t

    • [In] parameter

    • The size of the data

  • pCRCValue –

    • Type: size_t*

    • [Out] parameter

    • The calculated CRC value

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success
