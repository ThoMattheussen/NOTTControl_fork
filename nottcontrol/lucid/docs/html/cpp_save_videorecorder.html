VideoRecorder#

class VideoRecorder#

The video recorder facilitates the grouping of image data into video files.

Public Functions

VideoRecorder()#

An empty constructor.

The video recorder defaults to an empty video parameters object and ‘savedvideos/video<count>.avi’ for the file name pattern.

VideoRecorder(VideoParams params, const char *pFileNamePattern = "savedvideos/video<count>.avi")#

A constructor.

The video recorder defaults to different settings depending on the extension of the file name pattern:

  • ’.avi’ defaults to raw codec, BGR8 pixel format

  • ’.mov’ defaults to raw codec, RGB8 pixel format

  • ’.mp4’ defaults to H264 codec, BGR8 pixel format

Parameters:

  • params –

    • Type: VideoParams

    • Parameters of the video

  • pFileNamePattern –

    • Type: const char*

    • Default: “savedvideos/video<count>.avi”

    • File name pattern to use with this recorder

virtual ~VideoRecorder()#

A destructor.

virtual void SetRaw(uint64_t pixelFormat)#

SetRaw sets the recorder to write raw frames with no container or metadata. Frame data can be written for any pixel format, but data cannot be played back. It will also set the file extension to ‘.raw’.

Warning

  • changes file extension

  • cannot be called while recording

Parameters:

pixelFormat –

  • Type: uint64_t

  • Pixel format to use

Returns:

  • none

// prepare and open video recorder to prepare to create recorded video
{
       void RecordVideo(const std::vector<Arena::IImage*>& images)
       {
           Save::VideoParams params(images[0]->GetWidth(), images[0]->GetHeight(), 25);
           Save::VideoRecorder videoRecorder(params, Cpp_Record/video.mp4);

           videoRecorder.SetH264Mp4BGR8();

           videoRecorder.Open();
       }
}

  • changes file extension

  • cannot be called while recording

virtual void SetFileNamePattern(const char *pFileNamePattern)#

SetFileNamePattern sets a file name pattern. Changing the extension via this call will change the output container.

The video recorder defaults to different settings depending on the extension of the file name pattern:

  • ’.avi’ defaults to raw codec, BGR8 pixel format

  • ’.mov’ defaults to raw codec, RGB8 pixel format

  • ’.mp4’ defaults to H264 codec, BGR8 pixel

Warning

  • changes to the file extension will change the container accordingly

  • set file name pattern before setting other settings

  • cannot be called while recording

Parameters:

pFileNamePattern –

  • Type: const char*

  • File name pattern to set

Returns:

  • none
