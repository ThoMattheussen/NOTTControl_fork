acCategoryGetNumFeatures#

AC_API acCategoryGetNumFeatures(acNode hNode, size_t *pNumFeatures)#

acCategoryGetNumFeatures retrieves the number of features of a category node.

Parameters:

  • hNode –

    • Type: acNode

    • [In] parameter

    • A node

    • Accepts category nodes

  • pNumFeatures –

    • Type: size_t*

    • [Out] parameter

    • Number of features

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success

acCategoryGetFeature#

AC_API acCategoryGetFeature(acNode hNode, size_t index, acNode *hFeatureNode)#

acCategoryGetFeatureAndAccessMode retrieves a feature node by its index.

Parameters:

  • hNode –

    • Type: acNode

    • [In] parameter

    • A node

    • Accepts category nodes

  • index –

    • Type: size_t

    • [In] parameter

    • Index of the feature

  • hFeatureNode –

    • Type: acNode*

    • [Out] parameter

    • Feature node (node of any type)

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success

acCategoryGetFeatureAndAccessMode#

AC_API acCategoryGetFeatureAndAccessMode(acNode hNode, size_t index, acNode *hFeatureNode, AC_ACCESS_MODE *pAccessMode)#

acCategoryGetFeatureAndAccessMode retrieves a feature node and its access mode by its index.

Parameters:

  • hNode –

    • Type: acNode

    • [In] parameter

    • A node

    • Accepts category nodes

  • index –

    • Type: size_t

    • [In] parameter

    • Index of the feature node

  • hFeatureNode –

    • Type: acNode*

    • [Out] parameter

    • Feature node (node of any type)

  • pAccessMode –

    • Type: AC_ACCESS_MODE*

    • [Out] parameter

    • Feature node’s access mode

Returns:

  • Type: AC_ERROR

  • Error code for the function

  • Returns AC_ERR_SUCCESS (0) on success
